

1. 能使用tool，相当于人类有了手脚。 如：
	- 读写文件，网络搜索，读取网页...
	- 调用内部接口，如直接调用服务端/客户端暴露的接口
	  
2. 能使用skill，相当于可以仿照别人的实践经验。如：
	- 写个开场引导的规范流程skill，让agent按照skill的内容，引导新用户告知一些基础信息，如昵称、偏好等
	- 写个冷场破冰的最佳实践skill，让agent按照skill的内容，选择合适的方式主动开启新话题或玩文字游戏等
	- 能挂载知识库，注入企业内部的产品白皮书、操作手册、FAQ文档，让agent瞬间化身资深客服，给出100%符合企业规范的精准解答
	- 能角色扮演...
	  
3. 能拥有 memory，相当于有了长期记忆和个性。 如：
	- 它会将对话历史和重要偏好持久化在本地文件或数据库中，即使重启程序，它也记得你之前的约定（如“我不喜欢用 Markdown 格式回复”）
	- 它能通过检索历史对话，从过去的失败经验中“学习”，避免在同一个任务上反复犯错

4. 能主动聊天甚至干活，通过 cron & heartbeat 可以主动lian
