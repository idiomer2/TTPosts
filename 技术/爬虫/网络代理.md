## 服务器代理


1. **下载 + 解压 + 可执行 + 线路配置**
```bash
wget https://gh.llkk.cc/https://github.com/MetaCubeX/mihomo/releases/download/v1.19.20/mihomo-linux-amd64-compatible-v1.19.20.gz

gzip -d mihomo-linux-amd64-compatible-v1.19.20.gz
mv mihomo-linux-amd64-compatible-v1.19.20 /usr/local/bin/mihomo
chmod +x /usr/local/bin/mihomo

mkdir /etc/mihomo
cp config.yaml /etc/mihomo
```
2. **以守护进程的方式，运行 mihomo**
```bash
cat > /etc/systemd/system/mihomo.service << 'EOF'
[Unit]
Description=mihomo Daemon, Another Clash Kernel.
After=network.target NetworkManager.service systemd-networkd.service iwd.service

[Service]
Type=simple
LimitNPROC=500
LimitNOFILE=1000000
CapabilityBoundingSet=CAP_NET_ADMIN CAP_NET_RAW CAP_NET_BIND_SERVICE CAP_SYS_TIME CAP_SYS_PTRACE CAP_DAC_READ_SEARCH CAP_DAC_OVERRIDE
AmbientCapabilities=CAP_NET_ADMIN CAP_NET_RAW CAP_NET_BIND_SERVICE CAP_SYS_TIME CAP_SYS_PTRACE CAP_DAC_READ_SEARCH CAP_DAC_OVERRIDE
Restart=always
ExecStartPre=/usr/bin/sleep 1s
ExecStart=/usr/local/bin/mihomo -d /etc/mihomo
ExecReload=/bin/kill -HUP $MAINPID

[Install]
WantedBy=multi-user.target
EOF

systemctl daemon-reload  # 重新加载 systemd
systemctl enable mihomo  # 启用 mihomo 服务
systemctl start mihomo  # 立即启动 mihomo

# [可选]
systemctl reload mihomo  # 重新加载 mihomo
systemctl status mihomo  # 检查 mihomo 的运行状况
journalctl -u mihomo -o cat -f  # 检查 mihomo 的运行日志
```
3. **在命令行中使用代理**
	1. 临时性：`http_proxy=http://127.0.0.1:7890 https_proxy=http://127.0.0.1:7890 all_proxy=socks5://127.0.0.1:7890  curl -v google.com`
	2. 会话级：`export http_proxy=http://127.0.0.1:7890; export https_proxy=http://127.0.0.1:7890;  export all_proxy=socks5://127.0.0.1:7890;`
	3. 永久性：`\necho -e "export http_proxy=http://127.0.0.1:7890\nexport https_proxy=http://127.0.0.1:7890\nexport all_proxy=socks5://127.0.0.1:7890" >> ~/.bashrc`
